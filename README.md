Math Quiz Bot for Telegram
==========================

Simple math quiz for [Telegram](https://telegram.org/).

Building
--------

You need JDK >= 1.8 and maven >= 3

    $ mvn clean package

Running
-------

Set bot token in `application.properties` file.

Run:

    $ java -jar math-quiz-bot-1.0.jar
